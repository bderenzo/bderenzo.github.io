Parassh

A problem

Recently I needed to execute a few commands on a few servers. Commands took a long time to execute and I didn't want to waste my time waiting.

$ ssh server1 -p 2222 date +'%Y-%m-%d %H:%M:%S.%N'
2020-05-10 15:00:05.608639037
$ ssh server2 -p 2222 date +'%Y-%m-%d %H:%M:%S.%N'
2020-05-10 15:00:10.528002284
$ ssh server3 -p 2222 date +'%Y-%m-%d %H:%M:%S.%N'
2020-05-10 15:00:15.885338988

No trivial solution

Except to install external programs like:

  • parrallel,
  • pssh,
  • ...

There is no built-in solution.

A shell script

I wrote a little script to do that in shell.

$ parassh -o '-p 2222' server{1..3} -- date +'%Y-%m-%d %H:%M:%S.%N'
server3
2020-05-10 15:00:20.136820026
server1
2020-05-10 15:00:20.179848294
server2
2020-05-10 15:00:20.199682719

Total: 3 host(s), 1 second(s)

The script is avalaible on github.
