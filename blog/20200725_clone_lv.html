Clone LV

Recently, I face a problem with LVM logical volume. How to duplicate it easily? How to duplicate it mounted? Here are the solutions.

Live copy with snapshot

First solution, you can clone using snapshot. With this solution, the logical volume to be cloned can stay mounted.

  • Create a new lv
  • Create a snapshot
  • Copy snaphsot to new lv
lvcreate --name lv-clone --size SIZE vg
lvcreate --snapshot --name snap0 --size SIZE vg/lv
dd if=/dev/vg/snap0 of=/dev/vg/lv-clone bs=4M

Cold copy

Other solution, is to do an unmounted copy:

  • Create a new lv
  • Copy lv to new lv
lvcreate --name lv-clone --size SIZE vg
dd if=/dev/vg/lv of=/dev/vg/lv-clone bs=4M
